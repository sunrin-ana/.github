### ✨ 동아리 소개
**AnA는 15년 동안 유지된 선린인터넷고등학교 소속 전공동아리입니다.** <br/>
저희는 세상을 연결하고, 세상을 안전하게 하며, 세상을 발전시키는 동아리입니다! <br/>
